## 주요 라이브러리

1. **Zustand + Immer**
   - 상태 관리를 위해 사용
   - Immer를 통해 불변성을 쉽게 관리

2. **Tailwind CSS**
   - 유틸리티 클래스 기반 CSS 프레임워크

3. **Axios**
   - API 요청을 위해 사용

4. **React Router**
   - SPA 라우팅을 위해 사용

5. **Prettier**
   - 코드 스타일 통일을 위해 사용

6. **ESLint**
   - 코드 품질과 문법 오류를 체크

## 설치 방법

npm install
npm run dev
